# J4R Box Frontend

E-commerce platform for games, accessories, and in-game currency.

## Features
- Browse products with search and filters
- Shopping cart functionality
- User authentication
- Admin product management
- Responsive design

## Tech Stack
- HTML5
- CSS3 (Tailwind CDN)
- Vanilla JavaScript

## Backend Repository
https://github.com/JayRaido/j4r-box-backend

## Setup
Open `index.html` in a browser or deploy to Netlify.
